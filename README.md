# InputSystem Generator External Asset Postprocessor
Unity3d script that add to [InputSystem code generator](https://docs.unity3d.com/Packages/com.unity.inputsystem@1.19/manual/ActionAssets.html#type-safe-c-api-generation) ability to select external [InputActionAsset](https://docs.unity3d.com/Packages/com.unity.inputsystem@1.19/manual/ActionAssets.html), to use InputActionReferences simultaniously with code generation
